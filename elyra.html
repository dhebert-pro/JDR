Version avec tous les dés actuels.

Elyra est une elfe jeune et débutante, accordeuse de ruines.

Cette version permet de lancer tous les dés actuels : 3 dés de race, 2 dés de classe et 1 dé de personnalité.

Pour l'instant, les actions automatiques chargées depuis actions/race_actions.json ne concernent que les icônes de race.

Le fichier JSON ne contient pas de champ order. L'ordre du tableau sert d'ordre d'affichage.
